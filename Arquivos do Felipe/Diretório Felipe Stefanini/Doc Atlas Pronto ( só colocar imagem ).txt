			Gráfico - ACUMULADO ABANDONO

		Tipo de Gráfico : Pizza

	1 - Filtros dentro da área do "Gráfico"

 - Eixo x: abandono_tx ( real )

	2 - Filtros dentro da área de "Filtros do usuários"

 - data_ligacao ( Este Mês ) clicar em editar e ir ao passo 2
 - tipo de componentes ( Período Relativo ) - ( Caixa de Seleção Múltipla )

	3 - Escolher Valores : Caixa de opção - Mês

	4 - Selecionar Valores dentro da opção Mês:

 - Este Mês
 - Mês anterior
 - Últimos 3 meses
 - Últimos 6 meses

Especifique os valores do filtro padrão :
Digitar: Este Mês
